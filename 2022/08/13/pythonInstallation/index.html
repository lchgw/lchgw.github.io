python和Pip的压缩包版本安装

1. 下载python压缩包

在python官方网站(https://www.python.org/downloads/)下载python3.10.6的压缩包
Windows embeddable package (64-bit)

2. 下载pip压缩包

在pip网站(https://pypi.org/project/pip/)下载pip22.2.2压缩包:
pip-22.2.2.tar.gz (2.0 MB view hashes)

3. 解压缩

将两个压缩包解压至H:\目录下

5. 配置环境变量

进入系统环境变量设置，在PATH环境变量下添加python的安装路径(H:\python3.10.6)

6. 验证

win+r打开cmd 输入python --version验证
Python 3.10.6

7. 下载脚本

在https://bootstrap.pypa.io/get-pip.py此网站下载 get-pip.py :
打开网页，右键另存为，保存，随后将 get-pip.py 放入 get-pip 文件夹，与 python3.10.6放于同一目录下，打开cmd,跳转至get-pip文件夹中，输入
python get-pip.py
显示
Successfully installed pip-21.3.1 setuptools-60.2.0 wheel-0.37.1
(第二种方式：直接将 get-pip.py文件直接放入python3.10.6文件夹中，打开cmd,运行
python get-pip.py
出现
Successfully installed pip-21.3.1 setuptools-60.2.0 wheel-0.37.1
)

8. 查看检验

查看python文件夹中的Script文件夹中出现
pip.exe
pip3.10.exe
pip3.exe
wheel.exe

9. 修改配置

修改 H:\python3.10.6\python310.pth文件:
python310.zip
.
#新增Lib及Lib\site-packages的路径
H:\python\python-3.10.6\Lib
H:\python\python-3.10.6\Lib\site-packages
#

8. cmd验证

pip --version
pip 22.2.2 from H:\python\python-3.10.6\Lib\site-packages\pip (python 3.10)

原因是python310._pth文件修改了sys.path,而sys.path包含了python查找模块的所有目录比如python310.zip表示 python目录\python310.zip,
